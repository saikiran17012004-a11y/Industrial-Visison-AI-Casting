# Industrial-Visison-AI-Casting
CNN-based defect detection for metal casting using TensorFlow and OpenCV
🛠️ Project Overview

This project automates quality inspection for casting parts.

    Problem: Manual inspection is slow and subjective.

    Solution: A Convolutional Neural Network (CNN) that identifies blowholes and cracks with 95%+ accuracy.

    Tech: Python, TensorFlow, Keras, OpenCV.
